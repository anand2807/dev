# 🏛️ Renaissance Options Trading System

A sophisticated algorithmic options trading system for the Indian stock market, inspired by Renaissance Technologies' quantitative approach.

## 🎯 Overview

This system combines advanced quantitative techniques with real-time market analysis to identify profitable trading opportunities while maintaining strict risk controls.

## 🚀 Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run market analysis
python main.py --mode analysis

# Run backtesting
python main.py --mode backtest --start-date 2024-01-01 --end-date 2024-03-31

# Launch dashboard
python main.py --mode dashboard
```

## 🌟 Key Features

- **Advanced Market Data Integration** - NSE/BSE real-time data
- **Sophisticated Options Analytics** - Complete Black-Scholes implementation
- **Multi-Strategy Trading Engine** - 4 volatility-based strategies
- **Professional Portfolio Management** - Real-time risk management
- **Comprehensive Backtesting** - Historical simulation
- **Interactive Web Dashboard** - Real-time monitoring
